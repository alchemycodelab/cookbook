# Conditional

```js
if (true) {
  'true!'
} else {
  'else'
}
```

```js
if(false) {
  'false!'
} else {
  'else'
}
```

```js
const number = 5
if(number > 3) {
  'big number!'
} else {
  'small number'
}
```
